      Methods

      TestGetBitmap Tests if getBitmap method returns a bitmap object.
      TestInitialize Creates a new instance of the canvas before each test.
      TestReset Tests if Reset method resets the canvas to its default state. Default state is when the pen is at position (0, 0) and the pen colour is black.
      TestSet_InvalidDimensions Tests if Set method throws an exception when both the width and height of the canvas are invalid. Invalid dimensions are when they are less than or equal to zero.
      TestSet_InvalidHeight Tests if Set method throws an exception when the height of the canvas is invalid. Invalid height is when the height is less than or equal to zero.
      TestSet_InvalidWidth Tests if Set method throws an exception when the width of the canvas is invalid. Invalid width is when the width is less than or equal to zero.
      TestSet_ValidDimensions Tests if Set method sets the width and height of the canvas to valid values.
      TestSetColour_InvalidColour Tests if SetColour method throws an exception when the colour is invalid. Invalid colour is when any of the RGB values are less than zero or greater than 255.
      TestSetColour_ValidColour Tests if SetColour method sets the colour of the pen to a valid colour.
